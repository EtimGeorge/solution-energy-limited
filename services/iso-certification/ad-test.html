Parameter Reference

Parameter Description Example Values
source The platform where the ad is shown facebook, linkedin, twitter, instagram, google
campaign Marketing campaign identifier iso9001_nigeria, virtual_training, internal_audit
service The service to direct users to certification, training, audit
form Pre-select form type nigeria, international
ad_id Unique ad identifier vt2025, cert001, audit_lagos
utm_source Alternative to source (UTM format) instagram, newsletter, google
utm_medium Marketing medium (UTM format) social, email, cpc
utm_campaign Alternative to campaign (UTM format) training_lead_gen, june_audits
How to use in real campaigns:
When setting up actual social media ads, replace "index.html" with your actual domain name (e.g., "https://seesglobal.com"). The parameters remain the same.
